## 📦 Hướng dẫn cài đặt (Installation)
Hướng dẫn từng bước để chạy dự án dưới máy local:

1. Clone repository này về máy:
   ```bash
   git clone [Link-Repo]
   ```
2. Di chuyển vào thư mục dự án:
   ```bash
   git clone [Tên-Thư-Mục]
   ```
3. Cài đặt các thư viện cần thiết:
   ```bash
   npm install / pip install -r requirements.txt
   ```
4. Cài đặt các biến môi trường (Environment variables): Tạo file `.env` theo mẫu `.env.example`.
5. Chạy ứng dụng:
   ```bash
   npm start / python manage.py runserver
   ```
